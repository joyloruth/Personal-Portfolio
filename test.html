Full Stack E-commerce Burger Bistro

Tech Stack:

PHP,JavaScript, CSS, HTML, MySQL,

Full stack restaurant e-commerce website that allows patrons to add/remove items from cart, delete order, view order summary, register new account or login to existing account, and checkout

REST API that creates, retrieves, updates and removes items, users and order to and from MYSQL database

PHP sessions store visitor preferences as they login and navigate over multiple pages

Full screen page loader that animates as the page is loading

Google Maps API for location services and PayPal REST API for B2C customer payment transactions
